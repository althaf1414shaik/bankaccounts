## 🚀 Overview
A RESTful backend service for managing bank accounts. Supports account creation, deposits, withdrawals, fund transfers, and transaction history tracking. Built with Spring Boot and JPA for robust data persistence.

## ✨ Key Features
- **Account Management** (Create, view, update, and delete accounts).
- **Deposits & Withdrawals** (Secure transaction processing with balance validation).
- **Fund Transfers** (Transfer money between accounts with real-time updates).
- **Transaction History** (Complete audit log of all account activities).
- **Balance Inquiry** (Check current balance and account details).

## 🛠️ Tech Stack
- **Backend**: Java 8, Spring Boot 2.7.16, Spring Data JPA, Hibernate.
- **Database**: MySQL 8.0.
- **Build Tool**: Maven.
- **Other**: Lombok, Validation API.
